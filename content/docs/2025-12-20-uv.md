## UV

Install `uv` by `cargo`:

```sh
cargo install uv
```

Set up UV's images:

```txt
# ~/.config/uv/uv.toml
[[index]]
url = "https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple"
default = true
```

Install Python envs:

```sh
uv python install 3.10 3.11 3.12 3.13 3.14
```
